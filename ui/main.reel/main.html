  • a function called toX() that returns the string "AX"
  • a second function called toY() that returns the string "AY"

Bravo is a suclass of Alpha that:

  • overrides toX() with a function that calls this.super() and prefixes the results with the string "B"
  • overrides toY() with the same function (literally a pointer to the same function)

Charlie is a suclass of Bravo that

  • initially overrides no methods

Delta is a suclass of Charlie that

  • overrides toX() with a function that calls this.super() and prefixes the results with the string "D"
