Foo is a class that has:

  • a function called toString() that returns the string "FooString"
  • a second function called toRope() that returns the string "FooRope"
  • a third function called toCord() that returns the string "FooCord"

Bar is a class that has only:

  • a function called toString() that returns the string "BarString"

ExtendedFoo is a suclass of Foo that:

  • overrides toString() with a function that calls this.super() and prefixes the results with the string "Extended"
  • overrides toRope() with the same function (literally a pointer to the same function)
  • does not overrides toCord()

ExtendedBar is a suclass of Bar that overrides
toString() with the same function used
to override ExtendedFoo's toString() and toRope().

new ExtendedFoo().toRope() returns ""...
it should return "ExtendedFooRope".

ExtendedFoo.prototype.toString.call(new ExtendedBar()) returns ""...
it should return "ExtendedFooString".

On the other hand, ExtendedFoo.prototype.toCord.call(new ExtendedBar()) returns ""...
as it should.
