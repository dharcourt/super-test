Foo is a class that has:

  • a function called toString() that returns the string "FooString"
  • a second function called toRope() that returns the string "FooRope"

Bar is a class that has only:

  • a function called toString() that returns the string "BarString"

ExtendedFoo is a suclass of Foo that:

  • overrides toString() with a function that calls this.super() and prefixes the results with the string "Extended"
  • overrides toRope() with the same function (literally a pointer to the same function)

ExtendedBar is a suclass of Bar that

  • overrides toString() with the same function used to override ExtendedFoo's toString() and toRope() (literally a pointer to the same function)
  • overrides toRope() with a function that calls this.super() and prefixes the results with the string "EXTENDED-"

Calling new ExtendedFoo().toRope():

  • returns ""
  • should return "ExtendedFooRope"

Calling ExtendedFoo.prototype.toString.call(new ExtendedBar()):

  • returns ""
  • should return "ExtendedFooString"

Calling new ExtendedFoo().toString():

  • after calling new ExtendedFoo().toString()"
  • then setting ExtendedFoo.prototype.toString to ExtendedBar.prototype.toString
  • returns ""
  • should return "EXTENDED-FooString"
