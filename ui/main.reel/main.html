Foo is a class that has
a function called toString() that returns the string "FooString", and
a second function called toRope() that returns the string "FooRope".

Bar is a class that has only
a function called toString() that returns the string "BarString".

ExtendedFoo is a suclass of Foo that overrides
both toString() and toRope() with the same function,
a function that calls this.super() and prefixes the results with the string "Extended".

ExtendedBar is a suclass of Bar that overrides
toString() with the a function,
the one used to override ExtendedFoo's toString() and toRope().

new ExtendedFoo().toRope() returns ""... it should return "ExtendedFooRope".

ExtendedFoo.prototype.toString.call(new ExtendedBar()) returns ""... it should return "ExtendedFooString".
