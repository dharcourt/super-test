Alpha is a class that has:

  • a function called toX() that returns the string "XA"
  • a second function called toY() that returns the string "YA"

Bravo is a suclass of Alpha that:

  • overrides toX() with a function that calls this.super() and appends the string "B" to the results
  • overrides toY() with the same function (literally a pointer to the same function)

Charlie is a suclass of Bravo that

  • initially overrides no methods

Delta is a suclass of Charlie that

  • overrides toX() with a function that calls this.super() and appends the string "D" to the results

Issue 1, invalid heuristic part 1: Calling new Bravo().toY():

  • should return "YAB"
  • returns ""

Issue 2, invalid heuristic part 2: Calling Bravo.prototype.toX.call(new Delta()):

  • should return "XAB"
  • returns ""

Issue 2, invalid caching: Calling new Delta().toX():

  • after calling new Delta().toX() once
  • then setting Charlie.prototype.toX to a function that calls this.super() and appends the string "C" to the results
  • should return "XABCD"
  • returns ""
